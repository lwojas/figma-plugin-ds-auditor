How do I generate a Personal Access Token?

  1. From the file browser, click the account menu in the top-left corner and select Settings.
  2. Select the Security tab.
  3. Scroll to the Personal access tokens section, then click Generate new token.
  4. Enter a name for your new token, assign the scopes you want, and press Return / Enter. For a complete list of the scopes you can assign to your personal access token, see the developer documentation.
  5. Copy the token that is generated
